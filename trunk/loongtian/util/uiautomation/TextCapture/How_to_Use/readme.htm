How to use GetWordDemo
1. Register the Library

GetWord library must be registered before usage. On Windows Vista, register GetWord library according to the directions showed in Pic.1 and Pic.2.
On Windows 2000/XP/Server 2003, run register.bat to register the library directly.

2. Run GetWordDemo On Windows Vista

On Windows Vista, you need to run GetWordDemo as an administrator. (Pic.3). You can do this according to the following directions:
1. Right click on the file "GetWordDemo.exe".
2. Select the "Properties" item on the popup menu.
3. Select "Compatibility" tab on the popup dialog.
4. Check the item "Run this program as an administrator", and click "OK" button.

In fact, GetWordDemo still works correctly for most of the software even if  GetWordDemo does not run as an administrator. But for the software which run as an administrator, GetWordDemo could not capture any text from them because of the security reason of Vista. In this case, you need to run GetWordDemo as an administrator to capture text from these software.
